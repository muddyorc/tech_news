# Tech News

![Preview do projeto](./assets/imagem_projeto.png)
*Preview do portal de notícias com layout desenvolvido utilizando CSS Grid.*

Projeto desenvolvido como parte do curso **Full-Stack da Rocketseat**, com foco no estudo de **HTML5**, **CSS3** e no uso de técnicas avançadas de **CSS Grid** para a construção de layouts modernos e responsivos.


## Tecnologias Utilizadas
- **HTML5**
- **CSS3**


## Objetivo do Projeto
O objetivo deste projeto é praticar a construção de um portal de notícias aplicando conceitos avançados de **layout com CSS Grid**.  
Foram explorados organização de seções, criação de áreas responsivas, alinhamentos complexos e estruturação limpa do conteúdo, simulando a interface de um site profissional de notícias.
